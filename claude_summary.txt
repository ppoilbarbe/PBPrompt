PBPrompt — Implementation Summary
==================================
Version: 1.5.0
Last updated: 2026-04-20


Overview
--------
PBPrompt is a cross-platform PyQt5 desktop application for storing, searching,
and translating AI prompts.  The project was written 100% by Claude (Anthropic).

Primary storage: SQLite (.sqlite, WAL mode).
YAML: import/export only (via ruamel.yaml).
Translation: deep-translator back-ends + custom Reverso adapter.
i18n: gettext, 7 locales (en, fr, es, it, ru, vi, zh_CN).
Python: 3.11+.  Build: setuptools.  Entry point: pbprompt.__main__:main.
Authors: PBMou + Claude (Anthropic) — shown in About dialog.

Data layer (data.py)
--------------------
PROMPTS_COLUMNS = ("ai", "group", "name", "local", "english")
  Text columns only — single source of truth for SQL, YAML and display order.
PROMPTS_BLOB_COLUMNS = ("image", "thumbnail")
  Stored as SQLite BLOBs; image exported as Base64 in YAML; thumbnail not exported.

PromptEntry(dataclass)
  Fields: ai, group, name, image (bytes|None), thumbnail (bytes|None),
          local, english  (all str default ""; bytes default None)
  to_dict() → encodes image as base64 ASCII if present; omits None image
  from_dict() → decodes base64 image if present; ignores unknown keys

PromptCollection
  entries: list[PromptEntry]
  file_path: Path | None
  modified: bool

  SQLite I/O:
    load(path) → opens WAL-mode DB, checks "pbprompt" signature table
                  (raises ValueError if absent), auto-migrates missing columns,
                  reads all rows ORDER BY id
    save(path?) → CREATE TABLE IF NOT EXISTS + DELETE + INSERT + version stamp
    Migration: TEXT columns get NOT NULL DEFAULT ""; BLOB columns get no DEFAULT.

  YAML I/O:
    import_yaml(path, replace=False, thumbnail_factory=None) → tuple[int, int]
                  (imported_count, skipped_count)
                  Deduplication: rows whose (ai,group,name,local,english) matches
                  an existing entry are skipped (also within-file dedup in append mode).
                  thumbnail_factory: callable(bytes) → bytes | None, called for each image.
    export_yaml(path) → writes YAML 1.2 list of dicts; full image as base64; no thumbnail.

  Mutation helpers (called from Qt model):
    append(), remove_at(), remove_indices(), update_field()

SQLite schema:
  CREATE TABLE pbprompt (key TEXT PRIMARY KEY, value TEXT NOT NULL DEFAULT '')
    → key='version' stores the app version that last wrote the file
  CREATE TABLE prompts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    ai TEXT NOT NULL DEFAULT '', "group" TEXT NOT NULL DEFAULT '',
    name TEXT NOT NULL DEFAULT '',
    image BLOB, thumbnail BLOB,
    local TEXT NOT NULL DEFAULT '', english TEXT NOT NULL DEFAULT '')
  Note: "group" is always quoted (SQL keyword).  BLOB columns are nullable, no DEFAULT.
  PRAGMA journal_mode=WAL; row_factory = sqlite3.Row.


Configuration (config.py)
--------------------------
AppConfig(dataclass) — fields:
  display_language: str = ""        # "" = system default
  translation_language: str = ""    # "" = system default
  translation_service: str = "google"
  translation_api_key/app_id/app_secret/libretranslate_url: str
  log_level: str = "INFO"
  recent_files: list[str] = []
  recent_files_max: int = 10
  thumbnail_width: int = 64         # IMAGE column thumbnail width (px), range 16-512
  thumbnail_height: int = 64        # IMAGE column thumbnail height (px), range 16-512
  window_{x,y,width,height}: int | None = None  # None = first launch
  last_import_dir: str = ""         # last directory used in YAML import dialog
  last_export_dir: str = ""         # last directory used in YAML export dialog
  last_image_dir: str = ""          # last directory used in image load dialog
  column_filters: dict[str, str] = {}  # persisted per-column filter values
                                       # keys: "ai","group","name","local","english"
  image_viewer_zoom_max: int = 4       # max zoom for Fit mode, range 1-16
  image_viewer_zoom_step: int = 10     # zoom step in %, range 1-100
  image_store_keep_original: bool = True   # if True, images stored as-is
  image_store_max_width: int = 1920    # max width (px) before resizing, range 64-32000
  image_store_max_height: int = 1080   # max height (px) before resizing, range 64-32000

  load() → reads config.yaml via ruamel.yaml; validates all fields;
            unknown keys preserved in _extra (round-trip safe)
  save() → writes YAML 1.2; omits window_* if None; omits last_*_dir if empty;
            omits column_filters if all empty
  _opt_int(d, key, min_val) → returns None if absent or not int; min_val=100 for size

Config location:
  Linux:   ~/.config/pbprompt/config.yaml
  Windows: %APPDATA%\pbprompt\config.yaml
  macOS:   ~/Library/Application Support/pbprompt/config.yaml


GUI — MainWindow (main_window.py)
-----------------------------------
Class: MainWindow(QMainWindow, Ui_MainWindow)
Signature: __init__(config, initial_file=None, parent=None)

Key design points:

1. MRO shadowing of retranslateUi — CRITICAL:
   Python's MRO makes MainWindow.retranslateUi() shadow Ui_MainWindow.retranslateUi().
   setupUi() calls self.retranslateUi() → always dispatches to the override.
   Consequence: ALL setShortcut() calls in the generated ui_main_window.py are DEAD CODE
   and are never executed at runtime.
   Fix: all QKeySequence shortcuts are set inside MainWindow.retranslateUi() with a
   comment explaining the reason. They are re-applied on every language change.

2. Keyboard shortcuts (set in retranslateUi):
   Ctrl+N/O/S/Shift+S/W/Q, Ctrl+,, Ins, Ctrl+D, Del, F6, F7,
   Ctrl+I, Ctrl+Shift+I, Ctrl+E, Ctrl+R (Refresh Thumbnails), F1 (About)

3. Status bar tips (_update_status_tips()):
   Qt shows QAction.statusTip() in the status bar when hovering menu items/toolbar.
   _update_status_tips() iterates all findChildren(QAction), builds statusTip from
   toolTip() + shortcut().toString(), sets it.  Called at end of retranslateUi().

4. retranslateUi() override uses gettext via get_translate().
   self._ shorthand is set here and reused throughout.

5. _source_model (PromptTableModel) is created AFTER setupUi/retranslateUi, so
   set_header_labels() and set_header_tooltips() are called explicitly in __init__()
   after model creation (retranslateUi guard: hasattr(self, "_source_model")).

6. Startup file loading:
   - initial_file (CLI arg) → _open_initial_file()
   - else → _autoload_last_file() (last entry in recent_files config)

7. File operations:
   - _on_file_open(): QFileDialog filter for .sqlite files
   - _on_file_save_as(): adds .sqlite extension if missing
   - open_file(): catches ValueError separately → "Not a PBPrompt database" dialog
   - _on_import_yaml(replace): opens YAML dialog (default dir: last_import_dir),
                               calls import_yaml(), refreshes model
   - _on_export_yaml(): opens save dialog (default dir: last_export_dir),
                        calls export_yaml(), shows count in status

8. Translation: commit in-flight editor before translating via QApplication.focusWidget()
   + fw.clearFocus() (triggers FocusOut → commitData + closeEditor).
   Never use tableView.setFocus() (Qt cancels commit if target is ancestor of editor).

9. Sort state tracking:
   _sort_col (int) and _sort_order (Qt.SortOrder) are tracked explicitly in MainWindow
   to avoid relying on QHeaderView internal state (which may reset during
   QSortFilterProxyModel.sort() signal processing).
   _on_header_clicked(): non-sortable columns (IMAGE, LOCAL, ENGLISH) reset the visual
   sort indicator to the tracked active column rather than moving it.

10. Persistent filters:
    On startup, _connect_signals() restores config.column_filters into the filter
    QLineEdit widgets (which triggers set_filter via textChanged).
    On closeEvent, the current filter values are written back to config.column_filters
    and config.save() is called.

11. Image clear (multi-row):
    _on_image_clear() clears all selected rows (not just the context-menu row).
    Shows a QMessageBox.question confirmation dialog (default: No) with the row count.
    Shortcut: Backspace (not Del — Del is reserved for deleting rows).

File menu structure:
  New (Ctrl+N), Open (Ctrl+O), Recent Files →, ---,
  Save (Ctrl+S), Save As (Ctrl+Shift+S), ---,
  Import YAML → [Add entries (Ctrl+I), Replace all (Ctrl+Shift+I)],
  Export YAML (Ctrl+E), ---,
  Close (Ctrl+W), ---, Quit (Ctrl+Q)


GUI — image_utils.py
---------------------
detect_image_format(data: bytes) → "jpeg" | "png" | None
  Magic bytes: JPEG=b'\xff\xd8\xff', PNG=b'\x89PNG\r\n\x1a\n'
generate_thumbnail(image_data, width, height) → bytes | None
  QImage.scaled(KeepAspectRatio, SmoothTransformation), saved as PNG bytes.
resize_for_storage(data, max_width, max_height) → bytes
  Resizes proportionally if either dimension exceeds the limit; preserves format (JPEG/PNG).
  Returns data unchanged if already within bounds or if decoding fails.
pixmap_from_bytes(data) → QPixmap | None
qimage_to_bytes(image, fmt="PNG") → bytes | None
ImageViewDialog(QDialog) — scrollable full-image viewer, shown on double-click.
  Zoom toolbar: Fit / ×1 / Width / Height / +step% / −step%.
  Mode persisted via class-level _last_mode and _last_scale (across dialog instances).
  Mode buttons (Fit/×1/Width/Height) are checkable; MANUAL mode unchecks all.
  Shortcuts: 1 (×1), l/w translatable (width), h (height), +/− numpad (step zoom).
  max_zoom passed from config.image_viewer_zoom_max.
  zoom_step passed from config.image_viewer_zoom_step (% per step, default 10).
  Zoom to cursor (mouse wheel): eventFilter captures the image pixel fraction under the
    cursor before applying zoom, then repositions scrollbars so the same pixel stays
    under the cursor. Uses _img_label.mapFrom(viewport, pos) — reliable because
    QScrollArea::setWidget reparents the label onto the viewport. QScrollBar.setValue
    clamps automatically to [min,max] when geometry prevents exact centering.
Image file dialog: shows a live preview thumbnail of the selected file.
  Guard: max(size − 8, 1) to prevent zero-size crash in QPixmap.scaled().


GUI — Models (models.py)
--------------------------
Column(IntEnum): AI=0, GROUP=1, NAME=2, IMAGE=3, LOCAL=4, ENGLISH=5

PromptTableModel(QAbstractTableModel)
  Wraps PromptCollection; exposes data/setData/flags/headerData.
  headerData() handles Qt.DisplayRole and Qt.ToolTipRole.
  set_header_labels(list[str]) / set_header_tooltips(list[str]) — 6-element lists.
  Editable columns: all except IMAGE.  EditRole = DisplayRole.
  IMAGE column: data() returns QPixmap via Qt.DecorationRole (from thumbnail bytes).
  set_image(source_row, full_image, thumbnail) — sets both, emits dataChanged+DecorationRole.
  Signals:
    collection_modified = pyqtSignal()  — emitted when the data changes
    cell_copied = pyqtSignal(str)       — emitted after Ctrl+C copy

ImageDelegate(CurrentCellHighlightDelegate)
  Delegate for Column.IMAGE.
  paint(): draws thumbnail centred, or dashed placeholder if no image.
    Suppresses DecorationRole ghost image by clearing opt.decorationPixmap
    and the HasDecoration feature flag before calling drawControl.
  sizeHint(): returns (thumb_w+8, thumb_h+8).
  createEditor(): returns None (IMAGE not inline-editable).
  editorEvent(): consumes MouseButtonDblClick (view handles it).
  update_size(w, h): called after settings change.

MultiFilterProxyModel(QSortFilterProxyModel)
  Per-column regex filters.  filterAcceptsRow() applies all active patterns (AND).
  IMAGE column is skipped in filterAcceptsRow().
  lessThan() implements multi-key sort: converts Column to int() explicitly before
  calling model.index() (PyQt5/SIP bug: IntEnum may return invalid index otherwise).
  sort() no-ops for IMAGE/LOCAL/ENGLISH columns.

MultiLineDelegate(CurrentCellHighlightDelegate)
  In-place QPlainTextEdit for LOCAL and ENGLISH columns (col 4 and 5).
  eventFilter():
    Enter/Return alone → commitData + closeEditor (validates)
    Ctrl+Enter or Shift+Enter → insertPlainText("\n")
    Tab → EditNextItem, Backtab → EditPreviousItem
    Up/Down → natural cursor movement (return False)
  sizeHint(): uses view.columnWidth(col) for word-wrap height; fallback 100px if ≤ 8px.

CurrentCellHighlightDelegate(QStyledItemDelegate)
  Draws a 2px colored border around the current cell within a selected row.
  Color: palette Highlight darkened to 160%.

PromptTableView(QTableView)
  Custom subclass. setAcceptDrops(True), setDragDropMode(DropOnly).
  Handles Ctrl+C/X/V for clipboard operations.
  Context menu: dispatches to image menu for IMAGE col, otherwise Copy/Cut/Paste.
  Signals:
    image_activated = pyqtSignal(QModelIndex)       — double-click on IMAGE col
    image_drop_requested = pyqtSignal(object, object)  — drop: (source_idx, mime_data)
    image_load_requested = pyqtSignal(QModelIndex)  — Return/Enter key on IMAGE col
    image_paste_requested = pyqtSignal(QModelIndex) — Ctrl+V key on IMAGE col
    image_clear_requested = pyqtSignal(QModelIndex) — Backspace key on IMAGE col
  keyPressEvent(): handles image shortcuts (Return → load, Ctrl+V → paste,
    Backspace → clear) when IMAGE column is current.
  mouseDoubleClickEvent(): emits image_activated for IMAGE column.
  dragEnterEvent/dragMoveEvent/dropEvent: accept image data or local file URLs.

Row height management:
  showEvent() → QTimer.singleShot(0, resizeRowsToContents)  [after first layout]
  _on_section_resized(col, old, new) → resizeRowsToContents() for LOCAL/ENGLISH cols

  1. QIcon.fromTheme(theme_name) — FreeDesktop theme (Linux/KDE/GNOME)

  refresh_thumbnails → SP_BrowserReload

i18n (i18n.py)
--------------
get_translate() → returns the _ function for the active locale.
install_language(lang_code) → installs gettext for the given locale.
system_language() → detects OS locale (locale.getlocale(), fallback "en").
language_label(code) → "Native name (code)" e.g. "Français (fr)".
get_locale_dir() → Path to locales/, PyInstaller-aware (sys._MEIPASS).

Language change without restart:
  SettingsDialog saves new display_language → MainWindow calls retranslateUi().

Available locales: en, fr, es, it, ru, vi, zh_CN.


Translation back-ends
----------------------
BaseTranslator(ABC): to_english(text, source_lang), from_english(text, target_lang).
Exceptions are NOT caught internally; they propagate to _on_translate() in MainWindow,
which shows QMessageBox.critical with the exact error text.
Translation stops at first error; already-translated rows keep their results.

Supported services: google, mymemory, deepl, microsoft, yandex, libretranslate,
  baidu, papago, qcri, pons, linguee, reverso.


GUI — MainWindow image operations
-----------------------------------
_on_refresh_thumbnails(): regenerates all thumbnails using current thumbnail_width/height.
_on_image_activated(source_idx): opens ImageViewDialog for the entry's full image.
_on_image_drop(source_idx, mime_data): handles QMimeData image or local file URL.
_show_image_context_menu(pos, source_idx): image context menu. Checks has_image
  (entry.image is not None) to enable/disable the Copy, Save, and Clear actions.
  Menu structure:
    Load image from file…  [Enter]
    Paste image            [Ctrl+V]
    ──────────────────────────────
    Copy image             [Ctrl+C]   enabled only if image present
    Save image to file…              enabled only if image present
    ──────────────────────────────
    Clear image            [Backspace] enabled only if image present
_prepare_image_for_storage(data): calls resize_for_storage() if image_store_keep_original
  is False; called before every set_image() for file load, paste, and drop.
_on_image_load_from_file(source_idx): QFileDialog with preview (default: last_image_dir)
  → load image bytes → _prepare_image_for_storage() → set_image().
_on_image_paste(source_idx): QApplication.clipboard() → image() or text() (file path)
  → _prepare_image_for_storage() → set_image().
_on_image_copy_to_clipboard(source_idx): pixmap_from_bytes(entry.image) →
  QApplication.clipboard().setPixmap(pm). Copies the full stored image (not thumbnail).
_on_image_save_to_file(source_idx): detect_image_format() → filter + default_suffix
  ("jpg" for JPEG, "png" otherwise). Opens QFileDialog with setDefaultSuffix() so the
  extension is always appended. Writes raw bytes (no recompression). Updates last_image_dir.
_on_image_clear(source_idx): applies to all selected rows; confirmation dialog.
_on_import_yaml(): unpacks (imported, skipped) tuple; passes thumbnail_factory;
  updates last_import_dir.
_on_export_yaml(): updates last_export_dir.
_on_open_settings(): after settings change, calls _image_delegate.update_size() and
  tableView.setColumnWidth(Column.IMAGE, thumbnail_width + 12).
_on_context_menu(): checks column == IMAGE first, dispatches accordingly.
_on_header_clicked(): non-sortable columns reset the sort indicator to the active column.


Makefile notes
--------------

  MO_FILES computed via $(foreach lang,...) so all locales are covered.

Known limitations / open issues
---------------------------------
- Google Translate and Reverso use unofficial/undocumented endpoints.
- .mo files are not committed; run "make translations" after cloning.
- ui_*.py and resources_rc.py are not committed; run "make all" after cloning.
- deep-translator is not on conda-forge; pip install required.
- In conda environments: pip install -e . --no-deps to avoid reinstalling conda packages.
- The Duplicate toolbar button tooltip says "Duplicate the current row" but acts on
  the current selection (first selected row is duplicated if multiple are selected).
- Changing thumbnail size in Options does not auto-regenerate existing thumbnails;
  use "Refresh Thumbnails" from the Tools menu.
